# Fit4Me - Kişiye Özel Egzersiz Rehberi

Fit4Me, kullanıcıların BMI (Vücut Kitle İndeksi) değerlerine göre kişiselleştirilmiş egzersiz önerileri sunan bir web uygulamasıdır.

## Özellikler

- BMI Hesaplayıcı: Kilo ve boy bilgilerinizle anında BMI değerinizi hesaplayın
- Kişiselleştirilmiş Öneriler: BMI sonucunuza göre özel egzersiz, beslenme ve sağlık önerileri
- Detaylı Egzersiz Programları: Kardiyovasküler, güç, esneklik ve HIIT egzersizleri
- Responsive Tasarım: Mobil, tablet ve masaüstü cihazlarda mükemmel görünüm
- Modern Arayüz: Gradient renkler, animasyonlar ve kullanıcı dostu tasarım

## Kullanılan Teknolojiler

- HTML5: Semantik ve modern HTML yapısı
- CSS3: Gradient efektler, animasyonlar, flexbox, grid

## Kurulum

1. Projeyi bilgisayarınıza indirin:
```bash
git clone https://github.com/betulltok/fit4me-website.git
```

2. Proje klasörüne gidin:
```bash
cd fit4me-website
```

3. index.html dosyasını tarayıcınızda açın

## Kullanım

1. Anasayfa: Projenin genel tanıtımı ve özellikleri
2. BMI Hesaplama: Kilo ve boy bilgilerinizi girin, "Hesapla" butonuna tıklayın
3. Egzersizler: BMI sonucunuza göre önerilen egzersiz programlarını inceleyin

## BMI Kategorileri

- 18.5'in altı: Zayıf → Güç antrenmanları önerilir
- 18.5 - 24.9: Normal → Dengeli egzersiz programı
- 25.0 - 29.9: Fazla Kilolu → Kardiyovasküler egzersizler
- 30.0 ve üzeri: Obez → Hafif kardiyo ve profesyonel destek

## Öne Çıkan Özellikler

- Klavye Desteği: Enter tuşu ile hızlı geçiş ve hesaplama
- Animasyonlar: Smooth scroll, fade-in efektleri, hover animasyonları
- Mobil Uyumlu: Responsive tasarım ile tüm cihazlarda çalışır
- Akıllı Yönlendirme: BMI sonucuna göre otomatik egzersiz önerisi
