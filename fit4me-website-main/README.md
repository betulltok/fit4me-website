# Fit4Me – Sağlıklı Yaşam ve BMI Hesaplama Web Sitesi

Fit4Me, sağlıklı yaşam için, kullanıcıların temel fitness ve beslenme bilgilerine ulaşabildiği basit bir web sitesidir. Proje kapsamında kullanıcıların boy ve kilo bilgilerine göre Vücut Kitle İndeksi (BMI) hesaplaması yapılabilmektedir.

## Projenin Amacı

Bu projenin amacı:
- Sağlıklı yaşam, egzersiz ve beslenme konularında bilgilendirici bir web sitesi geliştirmek
- Kullanıcılara BMI hesaplama gibi temel bir sağlık aracı sunmak
- HTML, CSS ve JavaScript kullanarak fonksiyonel ve kullanıcı dostu bir arayüz oluşturmak

## Kullanılan Teknolojiler

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## BMI Hesaplama

BMI hesaplaması aşağıdaki formüle göre yapılmaktadır:

BMI = Kilo (kg) / (Boy (m) × Boy (m))

Hesaplanan değere göre kullanıcılar aşağıdaki kategorilerden birine atanır:
- Zayıf
- Normal
- Fazla Kilolu
- Obez

Her kategori için kısa bir bilgilendirme mesajı kullanıcıya sunulmaktadır.
